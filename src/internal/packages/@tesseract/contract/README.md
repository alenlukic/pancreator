# @tesseract/contract

Kind-agnostic contract wrapper types and structural checks for Tesseract Spec Contracts. The full normative schema lives in `/src/memory/handbook/contract-format.md`.

## Quickstart

From the repository root:

1. `pnpm install`
2. `pnpm --filter @tesseract/contract build`
3. `pnpm --filter @tesseract/contract typecheck`
4. `pnpm --filter @tesseract/contract test`
5. `pnpm --filter @tesseract/contract publint`
