# @daedaline/contract

Kind-agnostic contract wrapper types and structural checks for Daedaline Spec Contracts. The full normative schema lives in `/src/memory/handbook/contract-format.md`.

## Quickstart

From the repository root:

1. `pnpm install`
2. `pnpm --filter @daedaline/contract build`
3. `pnpm --filter @daedaline/contract typecheck`
4. `pnpm --filter @daedaline/contract test`
5. `pnpm --filter @daedaline/contract publint`
