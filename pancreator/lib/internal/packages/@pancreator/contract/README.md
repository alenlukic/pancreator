# @pancreator/contract

Kind-agnostic contract wrapper types and structural checks for Pancreator Spec Contracts. The full normative schema lives in `/lib/memory/handbook/contract-format.md`.

## Quickstart

From the repository root:

1. `pnpm install`
2. `pnpm --filter @pancreator/contract build`
3. `pnpm --filter @pancreator/contract typecheck`
4. `pnpm --filter @pancreator/contract test`
5. `pnpm --filter @pancreator/contract publint`
